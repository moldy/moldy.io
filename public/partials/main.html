An open source JavaScript model API for the browser + server that is actually worth using.

Installation

You’ll need Node.js installed ... obviously

Basics

Use moldy to create a data model with strongly typed properties to keep your data clean and santized

CRUD

For CRUD to work as you would expect, you need to `use` a moldy adapter like `moldy-ajax-adapter`

Adapters

Moldy uses a middleware layer for the CRUD feature to integrate adapters like `ajax`, `mongo` and `sockets` (write your own)

Browser + Server

That’s right! Finally a model library that makes it possible to share a single JSON schemas & use the same API in both worlds!
